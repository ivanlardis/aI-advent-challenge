# UI Smoke-сценарии (Chainlit + Playwright MCP)

**Цель:** протыкать ключевые пользовательские сценарии в браузере через Playwright MCP,
зафиксировать скриншоты и результат (прошёл/не прошёл).

**Предусловие:** запущен Chainlit на `http://localhost:8000`
(`chainlit run app.py --headless --port 8000`).

## Сценарии

### 1. Загрузка + приветствие
- Шаги:
  1. Открыть `http://localhost:8000`
  2. Дождаться welcome-сообщения
- Ожидание: в чате есть «Привет, Иван!» и список из 5 команд
  (`/compress`, `/summary`, `/dashboard`, `/profile`, `/reset`).
- Скриншот: `screenshots/01-welcome.png`

### 2. Команда /dashboard
- Шаги:
  1. В textbox ввести `/dashboard`, нажать Enter
- Ожидание: ответ ассистента содержит заголовок «Дашборд статистики»
  и текст «Пока нет данных. Начните общение!».
- Скриншот: `screenshots/02-dashboard.png`

### 3. Команда /summary
- Шаги:
  1. В textbox ввести `/summary`, Enter
- Ожидание: ответ «Пока нет данных по токенам.»
- Скриншот: `screenshots/03-summary.png`

### 4. Команда /profile (новая)
- Шаги:
  1. В textbox ввести `/profile`, Enter
- Ожидание: ответ «Профиль: Иван», «Загружено секций: 4»,
  список из 4 секций (Базовая информация / Цели и интересы /
  Стиль общения / Привычки и паттерны).
- Скриншот: `screenshots/04-profile.png`

### 5. Команда /reset (новая)
- Шаги:
  1. В textbox ввести `/reset`, Enter
- Ожидание: ответ «Сброшено. История и статистика очищены.»
- Скриншот: `screenshots/05-reset.png`
